# DataVault 🔐

Decentralized data ownership and privacy layer on OneChain. Store cryptographic proofs of your data on-chain while keeping actual data private.

## Deployed Contract

- **Package:** [`0x44ee8f5787aff3c7cd1f794eb31c4aedc291bf1d831183bb3eadd8742e656b2a`](https://onescan.cc/testnet/packageDetail?packageId=0x44ee8f5787aff3c7cd1f794eb31c4aedc291bf1d831183bb3eadd8742e656b2a)
- **Network:** OneChain Testnet

## Features

- Store SHA-256 hashes of files/data on-chain
- Auto-hash raw data in the browser (never leaves your device)
- Optional encrypted storage reference (IPFS, etc.)
- Delete entries you own
- Prove data ownership without revealing contents

## Tech Stack

- **Contract:** Move on OneChain
- **Frontend:** React + TypeScript + Vite + @mysten/dapp-kit
